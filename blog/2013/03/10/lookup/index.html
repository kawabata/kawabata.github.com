Emacs/Lookup で Wikipedia を見る

Mar 10th, 2013

Lookup の ndzim エージェント

Lookup はEmacs上で動作する、様々な辞書やリソースを横断して検索するツー ルです。最近、OpenZIMファイルを読むための ndzim エージェントを試作 したので紹介します。

OpenZIM フォーマット

OpenZIM フォーマットは、Webページやそれに付随する画像データを、圧縮し て１ファイルにまとめるフォーマットです。主に Wikipedia 等の MediaWiki の記事をアーカイブするために開発された、Wikipedia 財団公認のフォー マットです。

OpenZIM フォーマットの内部構造 は、LZMAで圧縮されたチャンクの集合体で す。OpenZIM形式にすることで、 日本語版Wikipedia 全記事（2011年現在） を画像込みで 7GByte、英語版Wikipediaを画像なしで12GByte 程度まで圧縮 します。様々な言語の Wikipeida 全記事の OpenZIMファイルがKiwixサイト にて配布 されています。

ZimLibのインストール

Lookup で OpenZIM ファイルを使うには、w3m と、zimlib に付属する zimreader/zimsearch ツールが必要です。一般にはgithubからダウンロード してmakeします。（要 liblzma ） 

% git clone https://gerrit.wikimedia.org/r/p/openzim.git
% cd cd openzim/zimlib
% ./autogen.sh
% ./configure
% make
% sudo make install

Lookup の設定

Lookup で OpenZIMを使うためには、 $HOME/.emacs.d/lookup/init.el に て、ZIMファイルを入れたディレクトリを検索エージェントで設定します。 

(setq lookup-search-agents
      '(...
        (ndzim "/path/to/dir")
       ))

上記のように設定すると、 /path/to/dir にある拡張子が.ZIMのファイル を検索エージェントが見つけます。

M-x lookup-restart にて、無事に辞書選択バッファに辞書が表示されたら、 辞書の上にカーソルを置いて、”F” を押して適当な単語で検索してみましょ う。検索結果はちゃんと表示されましたでしょうか？
